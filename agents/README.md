# EEN Agents

This folder defines the operating roles for Electrify Everything Now as a one-person, agent-assisted public-interest tools company.

Each agent file should answer:

- What the agent owns
- What inputs it uses
- What it outputs
- How often it runs
- What decisions it should surface to Josh
- What quality bar it must meet

- No retained photos unless explicitly opted in
- Direct Josh-style voice
- Useful, simple tools over fake precision
