# Trust Policy

Electrify Everything Now is an independent homeowner-first public-interest project.

## Core trust promise

```txt
Free, independent tools to help homeowners understand electrification projects before spending money.
```

## Required disclaimers

Use concise language near tools, photo upload, and result pages:

- EEN is not affiliated with Elephant Energy.
- EEN is not a contractor marketplace.
- EEN does not sell installations or steer users to a preferred contractor.
- EEN does not provide electrical, HVAC, legal, tax, rebate, or code advice.
- Tool results are educational planning guidance and should be verified with qualified professionals.
- Photos are used to help answer the question and should not be retained unless the user explicitly opts in.

## Product rules

1. Do not require email to see a result.
2. Do not imply certainty when the tool is only screening or triaging.
3. Do not collect project data that is not needed for the result.
4. Explain unknowns clearly.
5. Always include contractor questions or next verification steps.
6. Keep contractor resources secondary to homeowner usefulness.
7. Make photo upload optional, with manual fallback paths.

## Copy style

Good:

```txt
This does not tell you whether your panel is code-compliant. It helps you understand what to ask an electrician before assuming a full service upgrade is required.
```

Avoid:

```txt
You do not need a panel upgrade.
```

Better:

```txt
You may have a panel-space issue rather than a full service-capacity issue. Ask your electrician to confirm the actual constraint.
```
