# Electrify Everything Now Operating Plan

Electrify Everything Now is an independent, homeowner-first public-interest tool site. It helps people plan electrification projects before they hire anyone, pay for an upgrade, or get pushed into the wrong project.

## First 60-day audience

Primary audience: homeowners actively considering electrification projects:

- Heat pumps
- Heat pump water heaters
- Electrical panels and panel-upgrade avoidance
- Balcony solar
- EV charging
- Induction and other electrification projects

Contractors are a secondary audience. Contractor resources should help customers understand options, not steer leads.

## Commercial posture

For now, EEN should feel extremely independent:

- Not affiliated with Elephant Energy
- Not a contractor marketplace
- Not lead generation
- No paywall for core tools
- Email is optional
- Photos are not retained
- Guidance is educational, not electrical, HVAC, tax, legal, or code advice

## Optimization priorities

1. Traffic and distribution
2. Tool starts and completions
3. Feedback from real homeowners
4. Trust and usefulness
5. Email capture only as a secondary signal

## Core operating loop

```txt
Homeowner question
→ simple tool
→ plain-English result
→ contractor questions
→ related article
→ feedback
→ analytics
→ improvement issue
→ better tool
→ distribution
→ more users
```

## Voice

Josh-style direct voice with a public-interest feel:

- Plain English
- Direct answer first
- Helpful, not salesy
- Slightly contrarian when useful
- Careful about uncertainty
- Practical homeowner questions and next steps
